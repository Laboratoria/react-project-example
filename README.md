# react-project-base

React project base

## Stack

* Create React App
* Eslint Airbnb
* Prettier
* Yarn
* Redux
* redux-thunk
* react-intl - `es`, `pt-BR` and `en`
* Material-ui and Laboratoria-ui
* Husky - Git Hooks
* Fetch - HTTP Request

## Install

```shell
yarn
```

## Run

```shell
yarn start
```
